Map Specifications

All maps are in their own folder with 3 files:
1) A hit map named HitMap.png
	- All countries/territories should be a distinct RGB color - alpha channel is ignored.
	- Water (inaccessible areas) should be pure blue RGB (0,0,255) - alpha channel is ignored.
2) A display map named DisplayMap.png
	- The visible part of the map
	- Alpha channels can vary and will be displayed as indicated.
3) An information mapping file in JSON format named Properties.json
